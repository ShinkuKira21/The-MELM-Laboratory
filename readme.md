# MELM-Laboratory

This repository contains research, experiments, and demos for the **Modular Educative Language Model (MELM)**.

> **Note:** This is a research repository and not intended for production use.

## Repository Structure

- `documentation/` – Research papers and API documentation.
- `experiment/` – Experimental code, notebooks, and datasets.
- `showcase/` – Demo outputs, visualisations, and results.
